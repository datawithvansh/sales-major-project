# 📊 Vendor Performance Analysis Dashboard

An **End-to-End Sales & Vendor Analytics Project** featuring a **modern interactive web dashboard**, powered by **SQL analysis and Python-based ETL pipeline**.

This project simulates a **real-world business intelligence system** used by organizations to monitor vendor performance, track KPIs, and optimize sales and inventory decisions.

# 🎯 Business Objectives

This project is designed to solve key business problems:

* Track **Total Sales, Profit, and Margin**
* Identify **Top Performing Vendors**
* Monitor **Inventory & Unsold Stock**
* Analyze **Monthly Sales Trends**
* Evaluate **Freight Cost Efficiency**
* Compare **Brand & Category Performance**

```
cd vendor-performance-dashboard
```

## 2️⃣ Open Dashboard

Simply open:
